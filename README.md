## 💡 Genel Bilgi

**Python Müzik Çalar**, Pushbullet üzerinden kontrol edilebilen bir masaüstü müzik oynatıcısıdır.  
Müzik çalma, duraklatma, ses seviyesi değiştirme, bilgisayar kilitleme gibi işlemleri uzaktan yönetebilirsiniz.
